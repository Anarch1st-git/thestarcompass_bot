# Telegram Astro Bot

Telegram-бот для расчёта астрологических карт и отправки пользователю готовых интерпретаций.

## Возможности

- регистрация и обработка команд Telegram-бота;
- расчёт натальной карты;
- расчёт синастрии для двух людей;
- расчёт транзитной карты;
- генерация SVG/PNG-изображений астрологических карт;
- отправка результатов пользователю через Telegram Bot API;
- inline-клавиатуры и пошаговые диалоги;
- фоновые задачи через очередь Redis/RQ;
- интеграция оплаты подписки через ЮKassa.

## Технологии

- Python 3.10+
- python-telegram-bot
- Kerykeion
- Redis / RQ
- aiohttp
- CairoSVG
- ЮKassa API

## Структура проекта

```text
.
├── bot.py                 # точка входа Telegram-бота
├── astrocalc.py           # расчёт карт, аспектов и текстовых интерпретаций
├── natal_handler.py       # сценарий создания натальной карты
├── synastry_handler.py    # сценарий создания синастрии
├── transit_handler.py     # сценарий создания транзита
├── start_handler.py       # стартовый сценарий и главное меню
├── keyboards.py           # inline-клавиатуры
└── tasks.py               # фоновые задачи и отправка результатов
```

## Быстрый старт

1. Создайте виртуальное окружение:

```bash
python -m venv venv
source venv/bin/activate
```

Для Windows:

```bash
venv\Scripts\activate
```

2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Создайте файл `.env` на основе `.env.example` и заполните значения:

```bash
cp .env.example .env
```

4. Запустите Redis.

5. Запустите бота:

```bash
python bot.py
```

6. Запустите воркер фоновых задач, если используется очередь:

```bash
rq worker
```

## Настройки

Проект ожидает внешние настройки: токен Telegram-бота, параметры базы данных, Redis и платёжного провайдера. Не публикуйте реальные токены и ключи в репозитории.

Пример переменных окружения находится в `.env.example`.
